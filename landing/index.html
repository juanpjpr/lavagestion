Preguntas frecuentes

¿La app es gratis?

Sí, la descarga es gratis y tenés 14 días para probar todas las funciones sin límite. Después elegís un plan.

¿Solo funciona en celular?

No. Tenés la app Android para el día a día y un dashboard web para cuando quieras usar la computadora. Misma cuenta, mismos datos.

¿Necesito saber de tecnología?

Para nada. Si sabés usar WhatsApp, sabés usar LavaGestión. Está diseñada para ser ultra simple.

¿El WhatsApp al cliente tiene costo extra?

No, está incluido en el plan Profesional y Negocio. Se envía automáticamente cuando marcás un pedido como "Listo".

¿Y si no tengo Android?

Podés usar el dashboard web desde el navegador de cualquier celular, tablet o computadora. La app para iOS está en desarrollo.

¿Puedo cancelar cuando quiera?

Sí, sin penalidad. Cancelás desde tu cuenta con un toque. Sin llamadas ni trámites.
